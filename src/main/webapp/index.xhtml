jpa-demos

Demoprojekt zum Shortcut JPA 2.1

view/countryCrudDemo.xhtml Demo von Create/Read/Update/Delete view/orderDemo.xhtml Demo von Kaskadierung ond Orphan Removal view/default/personList.xhtml Demo einer Request-übergreifenen Datenerfassung (Variante mit 'normalem', TX-gebundenem EntityManager) view/appmngd/personList.xhtml Demo einer Request-übergreifenen Datenerfassung (Variante mit Applikations- und Konversations-gebundenem EntityManager) view/unsync/personList.xhtml Demo einer Request-übergreifenen Datenerfassung (Variante mit unsynchronisiertem und Konversations-gebundenem EntityManager)

Weitere Demos sind in diesem Projekt als SE-Demos programmiert. Sie finden sie als JUnit-Tests in src/test/java.
